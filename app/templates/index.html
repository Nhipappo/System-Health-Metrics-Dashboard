Доступные эндпоинты:

GET /health - Проверка здоровья
POST /items/ - Создать элемент
Body JSON: {"name": "test", "description": "описание"}
GET /items/ - Получить все элементы
GET /items/{id} - Получить элемент по ID
DELETE /items/{id} - Удалить элемент по ID
GET /metrics - Метрики Prometheus
GET /docs - Swagger документация
GET /redoc - ReDoc документация

Тестирование API:

Примеры curl команд:
# Создать элемент
curl -X POST "http://localhost:8000/items/" \
-H "Content-Type: application/json" \
-d '{"name":"Тест","description":"Описание"}'

# Получить все элементы
curl "http://localhost:8000/items/"

# Получить элемент по ID
curl "http://localhost:8000/items/1"
